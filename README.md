# YouTube Video Downloader by Nandan

YouTube Video Downloader by Nandan is an accessible NVDA add-on designed for screen reader users.

It allows users to download YouTube videos as MP4 or extract audio as MP3 easily using NVDA.

## Features

- Download YouTube videos as MP4
- Extract audio as MP3
- Automatic best quality download
- Background downloading
- NVDA speech progress updates
- Screen reader friendly interface

## How to Use

1. Copy a YouTube video link.
2. Press NVDA + Shift + Y.
3. Choose MP3 or MP4 download.

The download will start in the background.
